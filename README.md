## ✨ Overview

CodeWrapped transforms any public GitHub profile into a beautiful, story-driven experience.

Instead of displaying repositories and statistics as plain data, it presents your coding journey through interactive insights, developer analytics, language breakdowns, repository highlights, and a shareable profile — all in a clean, editorial interface inspired by modern product design.

---

## ✨ Features

- 🔍 Search any public GitHub profile
- 📖 Cinematic Developer Story
- 🧠 Developer Archetype
- 🧬 Coding DNA Analysis
- ⚡ Animated Power Score
- 📊 Repository & Language Insights
- 🏆 Achievement Highlights
- 🖼️ Canvas-based Share Card
- 📥 PNG Export
- 🌙 Light & Dark Theme
- 💾 Local Storage Cache
- 📱 Fully Responsive Design
- ♿ Accessible & Keyboard Friendly
- ⚠️ GitHub API Error Handling
