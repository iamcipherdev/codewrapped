# CodeWrapped

> **Your GitHub. Beautifully told.**

Transform any public GitHub profile into a cinematic developer story using **pure HTML, CSS, and Vanilla JavaScript**. Enter a GitHub username and instantly explore repositories, coding patterns, achievements, language insights, and a beautifully crafted shareable profile.

---

## ✨ Features

- 🎬 Cinematic developer story experience
- 🔍 Public GitHub username search
- 📊 Repository and language analysis
- 🧠 Developer archetype generation
- ⚡ Animated Power Score
- 📈 Coding DNA visualization
- 🏆 Achievement highlights
- 📅 Contribution timeline
- 🌙 Light & Dark mode
- 💾 Local caching & recent searches
- 📱 Fully responsive design
- 🖼️ PNG share card export
- 🔗 Share profile with URL
- ♿ Accessible keyboard navigation
- ⚡ Smooth 60 FPS animations
- 📦 Zero dependencies

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES Modules)
- GitHub REST API
- Canvas API
- LocalStorage

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/codewrapped.git
```

Open the project folder and serve it using any static server.

Example:

```bash
python -m http.server
```

or simply deploy to:

- GitHub Pages
- Netlify
- Vercel (Static)

No installation required.

No build step.

No dependencies.
