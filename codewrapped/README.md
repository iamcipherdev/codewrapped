# CodeWrapped

**Your GitHub. Beautifully Told.**

CodeWrapped transforms any public GitHub profile into a cinematic developer story. It presents a developer archetype, Power Score, coding DNA, language analysis, selected repositories, milestones, timeline, insights, and a downloadable share card.

## Run locally

Open `index.html` in a modern browser. For the most consistent browser module behavior, serve the folder with any simple static file server and visit its local address.

No installation, compilation, account, environment configuration, or remote data store is required. Profile information is requested directly from GitHub's public interface and cached in the browser for 30 minutes.

## Hosting

Upload the repository contents to any static host. The relative paths are compatible with project subdirectories.

## Privacy

CodeWrapped reads public information only. Theme preference, recent searches, and temporary profile cache stay in the visitor's browser.
